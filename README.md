IoT project using ESP32 and Android app to control and monitor devices in real time (LED &amp; push-button) with digital I/O.
# 📱 ESP32 IoT Control App

## 📌 Overview
This project demonstrates a simple IoT solution using an ESP32 microcontroller and an Android application to monitor and control devices in real time.

The system allows users to:
- Read digital input (push-button)
- Control LED output
- Monitor device status remotely

---

## 🧠 Technologies Used

- ESP32 (Microcontroller)
- Arduino IDE (C/C++)
- Android Studio (Kotlin)
- GPIO (Digital Input/Output)
- Wi-Fi / Bluetooth communication (optional)

---

## ⚙️ System Architecture

[Android App] ⇄ [ESP32] ⇄ [LED / Button]

---

## 🔌 Hardware Components

- ESP32
- LED
- Push-button
- Resistors
- Breadboard
- Jumper wires

---

## 💡 Features

- Real-time LED control
- Button state monitoring
- Simple and intuitive mobile interface
- Low-cost IoT solution
